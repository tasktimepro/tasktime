🚀 Application Features

Project Management

Create, edit, and delete projects with client information

Task Hierarchy

Create tasks and subtasks with unlimited nesting

Time Tracking

Start/pause/stop timers with one active timer at a time

Metrics & Analytics

View time spent and earnings across different periods

Invoice Generation

Generate professional PDF invoices with client details

Data Management

Export/import data as JSON for backup and migration

To fully test the application, try these actions:

  3. Start a timer on a task and verify only one timer can be active
  4. Check that other tasks are dimmed when a timer is running
  5. View metrics for different time periods
  6. Generate a PDF invoice
  7. Export data as JSON and import it back
